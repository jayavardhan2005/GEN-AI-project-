#  AI Student Performance Evaluator

A clever web application developed with **Machine Learning + Streamlit** to evaluate student performance and forecast:

*  Failure Risk
*  Chances of Placement
*  Performance insights by subject

--- ## Features

*  Choose a student from the CSV dataset
*  Use logistic regression to forecast failure
*  Use Random Forest to predict placement
* Interactive charts of performance
*  Calculating the probability of smart placement
*  AI-powered therapy guidance

--- ## Technology Stack

* Python
* Streamlit
* Scikit-Learn
* NumPy and Pandas
* Matplotlib
